# 神经元与神经网络

## 神经元

**Model** $ input \ x \rightarrow \text{neuron} \rightarrow \ output \ a$

- **a**  激活（activation，神经学术语，表示一个神经元向下游的其他发送了多少电脉冲输出）

## layer 层

一**组**输入**相同或相似**的神经元

- input layer 接收输入特征的一层
- output layer 输出最终结果的一层
